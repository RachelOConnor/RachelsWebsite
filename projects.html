HIVEBREAKER

Hivebreaker is an alien platformer shooter game in which the character Mason (aka you!) must fight to survive, while also trying to get as many kills as possible.
It's harder than it looks!

This game was made using C++ and Raylib, while the assets were created in Aseprite, Tiled, Photoshop, and Spine.

Hivebreaker recently won an award at Games Fleadh 2025.

Game published soon!

PATHS TO PEACE

Paths to Peace is a browser based game aiming to promote peace. Throughout the game, you will be given scenarios in which you must choose the correct answer that will avoid conflict.

This game was made using HTML, CSS and Javascript. I created all the assets using Aseprite.

Play the game here!

BRONZE AGE BRAWL

Bronze Age Brawl is a Hack N' Slash game where you play as Wilson, a daring and quirky caveman who battles for survival.

This game was made using C++ and SFML, while I used Aseprite to create the assets.

Bronze Age Brawl won "Best in Game Play" at Games Fleadh 2024.

Download the game here!

ERA SCIENCES

ERA Sciences specialises in delivering comprehensive compliance solutions for the life sciences, pharmaceutical, and medical device industries.

Over the summer, I updated this company's training platform to a more modern and sleek interface, as well as creating graphics and presentations for their online courses.

FLOWER POWER

This is a fun little project I've taken on within the last year to transform the inside of my car into it's own little flower garden!

As of now, only the front half of the car has been decked out in flowers, however there's still over 160 flowers here alone!

Each flower is made using crochet.

It's time consuming, but I'm loving the results!
